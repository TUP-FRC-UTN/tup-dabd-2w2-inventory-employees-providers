Información de la Lista de Proveedores

¿Qué es esta pantalla?

Esta pantalla te permite visualizar y gestionar los proveedores registrados. Cada sección está diseñada para ofrecer un acceso a la información de manera ordenada y sencilla.

Datos de Proveedores
  • Lista de Proveedores: Muestra los detalles de cada proveedor, incluyendo Nombre, CUIL, Servicio, Contacto, Dirección, Estado.
  • Búsqueda: Funcionalidad para filtrar la lista de proveedores, por proveedores activos y/o inactivos, o por otros filtros de búsqueda.
  • Limpiar: Permite limpiar los filtros seleccionados para la búsqueda.
  • Paginación: Control para navegar entre múltiples páginas de proveedores.
  • Nuevo: Permite ingresar un nuevo proveedor, nos redireccionará a un formulario para completarlo.
  • Exportar: Botones para exportar la lista a formatos PDF o Excel.
Acciones Disponibles
  • Editar: Nos redireccionará a un formulario para editar el proveedor seleccionado.
  • Eliminar: Nos solicitará confirmación para eliminar el proveedor seleccionado.
Nota:

Asegúrate de verificar la información antes de modificar, ingresar o eliminar un proveedor.
