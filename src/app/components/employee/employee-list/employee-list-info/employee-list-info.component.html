Información de la Lista de Empleados

¿Qué es este listado?

Este listado te permite visualizar y gestionar los empleados registrados. Cada sección está diseñada para ofrecer un acceso a la información de manera ordenada y sencilla.

Campos Principales
  • Empleado: Nombre y apellido de cada empleado.
  • Rol: El rol que cumple el empleado dentro del consorcio.
  • Documento: Documento de indentificación propio del empleado.
  • Fecha Contratación: Fecha de contratación del empleado.
  • Estado: Indica si el empleado está activo o fue eliminado.
Acciones Disponibles
  • Editar: Editar los información del empleado seleccionado.
  • Eliminar Eliminar el empleado seleccionado.
  • Visualizar: Visualizar la información del empleado seleccionado.
Nota:

Asegúrate de verificar la información antes de modificar, ingresar o eliminar un empleado.
